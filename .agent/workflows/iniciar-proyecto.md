# Flujo: Iniciar Proyecto Web Responsive

## Pasos

1. Confirmar el stack tecnológico consultando el skill `monitor-stack`
2. Crear la estructura base de carpetas:
   - `src/` - Código fuente principal
   - `src/components/` - Componentes reutilizables
   - `src/pages/` - Páginas o vistas principales
   - `src/utils/` - Funciones helper y logs
   - `src/styles/` - CSS y estilos
   - `assets/` - Imágenes, iconos, fuentes
   - `docs/` - Documentación del proyecto

// turbo
3. Crear archivo `src/utils/logger.js` con sistema de logs

4. Crear archivo `src/styles/variables.css` con breakpoints responsive

5. Crear `index.html` base con meta viewport para responsive

6. Documentar decisiones en `docs/decisions.md`
