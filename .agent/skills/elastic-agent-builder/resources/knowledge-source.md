## Resumen de Contenido

Elastic Agent Builder combina LLMs (Large Language Models) con características nativas de Elastic para ingeniería de prompts, contexto y automatización. 

### Conceptos Clave
1. **Agent Chat**: Interfaz de chat síncrona para interactuar con los agentes mediante lenguaje natural.
2. **Agents**: Impulsados por instrucciones personalizadas de LLM y la habilidad de usar herramientas (tools) para responder preguntas, tomar acción o soportar workflows.
3. **Tools**: Funciones modulares y reutilizables que los agentes usan para buscar, recuperar y manipular datos en Elasticsearch. Permite conectar las capacidades del agente a los datos.

### Capacidades Destacadas
- Relevancia y seguridad de Elasticsearch.
- Agentes y herramientas integrados o personalizados.
- Soporte de Model Context Protocol (MCP) y servidores A2A para exponer tools a clientes externos.
- Integración con Elastic Workflows (agentes pueden disparar workflows y workflows pueden invocar agentes).
- Interfaces programáticas (APIs de Kibana).
- Selección de modelos (pre-configurados o conectores LLM locales).
- Control de acceso por roles y API keys.
